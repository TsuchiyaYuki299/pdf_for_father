かんたんPDF書き込み・編集ツール

文字やふりがな（ルビ）、図形・直線の書き込み、移動、サイズや透明度の調整、背景色に合わせた文字消しがスムーズに行えます

PDFファイルをここにドラッグ＆ドロップ

または、ここをクリックしてパソコンからファイルを選択してください

編集ツールの選択

ペンの色 / 文字 / 直線 / 塗りつぶしの色 黒
太さ / 文字サイズ 12 px
不透明度 100 %
便利なキーボード操作
  • Ctrl + C : コピー
  • Ctrl + V : 貼り付け
  • Ctrl + Z : 元に戻す
  • Delete : 選択要素を消去
  • Shift + ドラッグ: 45度スナップ / 複数選択
